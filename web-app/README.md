# ⛳ Couple's Golf Diary (React + Tailwind for Vercel)

A clean, bright-themed fully structured web application version of **Couple's Golf Diary** ready for immediate Vercel deployment!

## ✨ Features Included

1. **English Interface & Bright Theme**: Redesigned beautifully using emerald primary accents, light sand-coloured card backdrops, and plenty of margins.
2. **Advanced Golf Course Options**:
   - Save the **Full Address** (displayed underneath the course title).
   - Configure custom **Total Course Par**.
   - Input separate **Course Rating (CR)** & **Slope** details for **Lady Tees** and **Blue Tees**.
3. **Simulated Tracker**: Easy interactive location helper.
4. **Local Storage Auto-Save**: Safely persists golf courses, scorecards, and uploaded photos indefinitely inside your phone or web browser!

## 🚀 How to Deploy directly on Vercel (Easiest Method)

### Method 1: Deploy with a connected GitHub Repository (Recommended)

1. Export this workspace by clicking **"Export as ZIP / Download ZIP"** in the top-right settings of the AI Studio UI, or push directly to your github repository.
2. Extract the ZIP and open the `/web-app` folder on your computer.
3. Push ONLY the contents of the `/web-app` folder to your own GitHub repository.
4. Log into [Vercel](https://vercel.com).
5. Click **"Add New..."** ➔ **"Project"** ➔ Import your newly created GitHub repository.
6. Under **Build & Development Settings**, Vercel will automatically detect `create-react-app` architecture. Click **"Deploy"**.
7. Your app is live! Open it on your phone, click "Add to Home Screen" to install it exactly like a native app.

## 💻 Local Workspace Development

To test or customize this web application locally:

```bash
# Navigate to the web application folder
cd web-app

# Install package dependencies
npm install

# Run the local development server
npm start
```
